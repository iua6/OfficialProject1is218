The Calculator Program



How it demonstrates OOP(Object-Oriented-Programming) principles:


The basic principles of OOP involves Abstraction, Encapsulation, Inheritance, and Polymorphism. There are also objects and classes. Together, they stand as the working principle of any object-oriented programming language.The calculator program demonstrates abstraction because the calculator allows you to call seamless programs and functions by just knowing what method to call and what parameters to input. Secondly, the calculator program demonstrates encapsulation when defining classes.Thirdly, the calculator programs exemplifies inheritance because it creates new classes that share the same information as past existing classes. Finally, the calculator program exhibits Polymorphism because it uses a reference to a parent class to affect an object in the child class. We might create a class called “horse” by extending the “animal” class. That class might also implement the “professional racing” class. The “horse” class is “polymorphic,” since it inherits attributes of both the “animal” and “professional racing” class.
